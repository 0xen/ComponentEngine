All assets were modeled by SpeedTree - http://www.speedtree.com/

Textures (compressed .DDS) designed for GGX-based PBR material system with the following convention: 
- BaseColor
	RGB channels: BaseColor value
	Alpha channel: Opacity

- Specular:
	Red channel: Occlusion
	Green channel: Roughness
	Blue channel: Metalness

- Normal (DirectX)
